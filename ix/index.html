MSR Virtual Internet Exchange

紹介 | Introduce

IPv6 IXは、中立的なBGP実験通信および学習プラットフォームとして主に使用されます。

この交換センターはIPv6のみをサポートしています。

私たちはMTU1500のみをサポートしています。

The IPv6 IX will be primarily used as a neutral BGP experiment communication and learning platform.

This exchange only supports IPv6.

We only support MTU1500.
